# agentic-dev

A tutorial repository for building with LangChain, LangSmith, and AI agents.

## Overview

This repository contains educational material and examples covering:

- **LangChain** - Building applications with large language models
- **LangSmith** - Monitoring and debugging LLM applications
- **LLM Monitoring** - Observability, tracing, and performance optimization
- **AI Agents** - Creating autonomous agents that can reason and take actions
